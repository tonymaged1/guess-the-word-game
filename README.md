# Guess the Word Game 🎮

لعبة تخمين الكلمات تفاعلية وممتعة تم بناؤها بالكامل باستخدام واجهات الويب الأساسية (Vanilla JS) وبدون الاعتماد على أي مكتبات خارجية لتوضيح مهارات التعامل المباشر مع المتصفح.

## 🚀 التقنيات المستخدمة (Tech Stack)
* **HTML5** - لبناء هيكل واجهة المستخدم.
* **CSS3** - للتصميم والتنسيق مع استخدام الـ Nesting والـ CSS Variables لمرونة الألوان.
* **JavaScript (ES6+)** - للتحكم الكامل في منطق اللعبة والتفاعل مع الـ DOM.

## ✨ المميزات الرئيسيّة (Key Features)
* **توليد ديناميكي لعناصر الصفحة (Dynamic DOM Generation):** يتم إنشاء مربعات الإدخال (Inputs) برمجياً بالكامل بناءً على عدد المحاولات (Tries) وطول الكلمة المراد تخمينها، مما يقلل حجم ملف الـ HTML.
* **تجربة مستخدم سلسة (Advanced UX):** دعم كامل للتنقل المتقدم والمريح داخل مربعات الحروف باستخدام الأسهم (`ArrowLeft` و `ArrowRight`).
* **إدارة ذكية لزر الحذف (Custom Backspace Behavior):** كود مخصص للتحكم في الأحداث عند الضغط على الـ Backspace لمسح الحرف الحالي والرجوع تلقائياً للمربع السابق لزيادة سلاسة اللعب.
* **نظام تلميحات ذكي (Smart Hint System):** يتيح مساعدة اللاعب عبر تصفية وفلترة المصفوفات (`Array.from`) لملء حروف عشوائية صحيحة في المربعات الفارغة المتاحة فقط والتي لم يتم إغلاقها.
* **مؤشرات بصرية ملونة (Visual Feedback):** تلوين المربعات برمجياً لإرشاد اللاعب بناءً على حالة الحرف:
  * 🟩 الأخضر (في الكود المعتمد): حرف صحيح في المكان الصحيح.
  * 🟧 البرتقالي (في الكود المعتمد): حرف صحيح ولكن في المكان الخطأ.
  * ⬛ الرمادي الداكن: حرف خاطئ تماماً.
